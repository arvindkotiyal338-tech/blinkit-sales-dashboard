# Blinkit Sales Analysis Dashboard

## 📊 Overview
This project analyzes retail sales data to uncover insights related to product performance, outlet types, and customer behavior.

## 🛠 Tools Used
- Excel
- Pivot Tables
- Data Cleaning
- Data Visualization

## 📈 Key Insights
- Generated $1.2M revenue across 8,523 orders
- Low-fat products contributed 65% of total sales
- Tier 3 cities generated highest revenue
- Medium outlets contributed 42% of sales
